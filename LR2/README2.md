# Реализация медианного фильтра на GPU с использованием текстурной памяти

## Описание работы программы

Данная программа реализует **9-точечный (3×3) медианный фильтр** для удаления шума «соль и перец» на изображении в оттенках серого. Основная особенность – использование **текстурной памяти CUDA** для доступа к пикселям, что обеспечивает аппаратное кэширование и автоматическую обработку границ (режим `CLAMP`).

---

## Основные этапы работы

### Загрузка изображения
Из файла `Lena.bmp` считывается изображение в оттенках серого, приводится к размеру **255×255** (как в исходном задании).

### Добавление шума «соль и перец»
Функция `add_salt_pepper` случайным образом заменяет **10%** пикселей на белые (255) или чёрные (0). Шум создаётся с помощью векторизованных операций NumPy для ускорения.

### Подготовка данных для GPU
- Изображение дополняется по краям (режим `edge`) для корректной обработки граничных пикселей.
- Данные преобразуются в `float32` и копируются в GPU-память.
- Создаётся **CUDA-массив** (`CUDAarray`) с форматом 32-битных чисел с плавающей точкой.
- Формируется **текстурный объект** (`cudaTextureObject_t`) с параметрами:
  - `addressMode = CLAMP` – граничные координаты заменяются ближайшими пикселями.
  - `filterMode = POINT` – без интерполяции (сохраняется точность).
  - `readMode = ElementType` – чтение как есть.

### Запуск CUDA-ядра
Ядро `medianFilter3x3` выполняется на GPU:
- Каждый поток обрабатывает один выходной пиксель.
- Через текстуру (`tex2D<float>`) извлекаются 9 соседних пикселей (3×3 окно).
- Собранные значения сортируются **пузырьковым методом**.
- Медиана (5-й элемент после сортировки) записывается в выходной массив.

### Измерение времени
Используются события CUDA (`cp.cuda.Event`).

### Сохранение результата
Отфильтрованное изображение (тип `uint8`) сохраняется как `filtered_image.bmp`, а зашумлённое – как `noisy_image.bmp`.

---

### Визуализация

Программа отображает три изображения:
1. Исходное (чистое).
2. Зашумлённое («соль и перец»).
3. Отфильтрованное (после медианного фильтра).

## Заключение

Программа реализует медианный фильтр для подавления импульсного шума с использованием текстурной памяти CUDA. Благодаря параллельной обработке каждого пикселя достигнуто высокое быстродействие – **менее 0.1 миллисекунды** для изображения 255×255. Полученные файлы сохраняются в формате BMP, а время выполнения выводится в консоль.
